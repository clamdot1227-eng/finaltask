I這是一個基於 Python Tkinter 開發的理財程式，旨在幫助使用者記錄每日收支、分析財務狀況並設定理財目標。

## 🚀 執行環境需求 (Prerequisites)

在一台全新的電腦上執行此專案前，請務必符合以下條件：

### 1. 作業系統要求 (System Requirement)
* **必須使用 Windows 作業系統**。
* **原因**：系統內部的統計圖表預設使用「微軟正黑體 (Microsoft JhengHei)」進行顯示。為了確保中文字體能正常呈現而不出現亂碼（方塊），請於 Windows 環境下執行。

### 2. 安裝 Python
* 請確保電腦已安裝 **Python 3.9.13** 版本。
* **注意**：安裝時請務必勾選「Add Python to PATH」選項。

### 3. 安裝必要套件
本系統使用了 `Matplotlib` 進行視覺化圖表繪製，請開啟終端機 (CMD 或 PowerShell) 並執行以下指令：
```bash
pip install matplotlib
```
### 4. 開啟方式
點擊app.py後右鍵，將其用Python開啟
若是在Visual Code中開啟，請打開整個資料夾才能順利執行
